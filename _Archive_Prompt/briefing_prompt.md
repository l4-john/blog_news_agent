STEP 1. 기사 불러오기
notion-search로 오늘 수집 일자 기준 기사 조회.
(
  data_source_url: collection://ae7e2fac-80f3-4ccc-928e-5ff059a854c9,
  query: "오늘의 내 코멘트",
  filters: { created_date_range: { start_date: "KST 오늘 날짜 - 1일 (YYYY-MM-DD)", end_date: "KST 오늘 날짜 (YYYY-MM-DD)" } }
)
예시: 오늘이 2026-05-22이면 → start: 2026-05-21, end: 2026-05-22
브리핑 포함 여부가 체크된 기사만 대상.
조회 필드: 제목, 브리핑 순서, 요약, 분야만. 나머지 불러오지 말 것.
불러온 기사 수를 대화창에 보고.
브리핑 순서 1번은 반드시 미국 증시 마감 시황(3대 지수 포함) 고정. 없으면 보고 후 2번부터 진행.

STEP 3. 오타 수정
저장된 브리핑 페이지 제목만 조회.
처리 순서: 제목만 조회 → 수정 → 저장 → 다음
검토 방식: 제목을 읽고 수정 항목이 있을 경우만 "총 M개 수정" 형식으로 보고. 수정 없으면 보고 생략.
