## 프로젝트 목적

네이버 블로그에 생활 용품 리뷰 포스팅을 자동 생성하여 네이버 애드포스트 심사 통과를 달성한다.
현재 블로그는 저품질 의심 상태이므로, 독창성·정보성·자연스러운 문체를 최우선 기준으로 삼는다.

---

## 워크플로우 구조

사용자 (상품명 + 실사용 경험 메모 입력)
       ↓
product-review-search
  웹 검색 → 스펙·가격·실구매 후기·경쟁 상품 수집
  실경험 메모 제공 시 팩트 시트 말미에 [사용자 경험 메모] 섹션으로 포함
  출력: 팩트 시트 (스펙/가격/장단점 키워드/경쟁 상품/SEO 키워드 2~3개 + 경험 메모)
       ↓  ← 팩트 시트 전문(경험 메모 포함)을 그대로 전달
product-review-writer
  MZ 캐주얼 1인칭 경험담 스타일, 2,000자 이상 본문 작성
  출력: 완성 본문 (제목 없음, 이미지 플레이스홀더 포함)
       ↓  ← 완성 본문 전문을 그대로 전달
product-review-saver
  애드포스트 체크리스트 검수 → SEO 메타데이터 생성 → 노션 Blog/Review/{포스팅 제목} 저장

---

## 에이전트 간 데이터 전달 규칙

- 오케스트레이터(Claude)가 중계자 역할을 한다. 각 에이전트의 출력 전문을 다음 에이전트의 입력으로 그대로 전달한다.
- search 결과(팩트 시트)는 writer 프롬프트 앞에 전문 첨부.
- writer 결과(완성 본문)는 saver 프롬프트 앞에 전문 첨부.
- 중간 결과를 요약하거나 편집하지 말 것. 원본 그대로 전달.

---

## 에러 처리 및 중단 조건

- search 단계: 검색 결과가 충분하지 않은 경우(신상품, 해외직구 상품 등으로 국내 후기 5개 미만) → 사용자에게 상황을 알리고 "그래도 진행할까요?" 확인 후 진행 여부 결정.
- writer 단계: 초안 완성 후 공백 제외 글자 수가 1,800자 미만이면 → 자동으로 보완하여 2,000자 이상으로 재작성. 사용자에게 별도 확인 불필요.
- saver 단계: 노션 저장 실패 시(API 오류 등) → 완성 본문을 채팅창에 그대로 출력하여 사용자가 직접 복사할 수 있도록 안내.

## 애드포스트 심사 핵심 원칙

1. 독창성 최우선 — 타 플랫폼 글 재탕·스피닝 절대 금지. 새 구조로 창작.
2. 솔직한 단점 포함 — 장점만 나열 시 광고성 어뷰징 의심받음. 단점 최소 1개 필수.
3. 2,000자 이상 — 공백 제외 기준. 얕은 정보성 글은 심사 탈락 원인.
4. 경험담 1인칭 서술 — "저도 써봤는데~", "실제로 해보니~" 형식으로 자연스럽게.
5. 리스트 최소화 — 개조식 나열이 전체의 20%를 넘으면 AI 생성 의심 신호.
6. 이미지 직접 촬영 필수 — 포스팅당 최소 3장. 📷 플레이스홀더 위치에 직접 찍은 사진 삽입.
7. Footer 항상 포함 — "개인 후기" 명시 + 이웃 추가 유도 문구.

## 에이전트 실행 방법

아래 네 가지 방식 모두 가능합니다.

기본형 (상품명만)
  "다이슨 에어랩 리뷰 포스팅 작성해줘"
  → 실사용 경험 없이 웹 수집 데이터만으로 작성. 경험담 부분은 일반적인 실구매자 반응 기반으로 작성.

상세형 (상품명 + 실경험 메모) ← 권장
  "쿠팡 에어프라이어 리뷰 써줘. 3달 전에 샀는데 예열이 필요 없어서 편하고, 기름 없이 고구마 구워먹는 게 진짜 맛있음. 근데 소음이 좀 있고 내부 코팅이 살짝 걱정됨."
  → 사용자 실경험을 1인칭 경험담에 직접 반영. 애드포스트 독창성 점수 대폭 향상.

사진 목록 추가형 (상품명 + 실경험 메모 + 보유 사진 목록) ← 최권장
  "쿠팡 에어프라이어 리뷰 써줘. 3달 전에 샀는데... 보유 사진: 제품 정면샷, 실제 사용 중인 모습, 음식 완성샷"
  → writer가 보유 사진 목록 기준으로만 📷 플레이스홀더 작성. 없는 사진 구도 요구 안 함.

촬영 예정형 (상품명 + 실경험 메모 + 직접 촬영 예정 명시)
  "쿠팡 에어프라이어 리뷰 써줘. 3달 전에 샀는데... 사진은 직접 촬영 예정."
  → writer가 글의 구조에 맞춰 찍으면 좋을 사진 구도를 📷 플레이스홀더로 안내. 포스팅 완성 후 안내된 구도대로 촬영해서 해당 위치에 삽입.

오케스트레이터가 product-review-search → product-review-writer → product-review-saver 순서로 자동 실행합니다.

## 주의사항

- 가능하면 직접 구매·사용한 상품을 리뷰하는 것을 권장. 실경험 없이 작성 시 일반 구매자 후기 기반으로 작성. (협찬·광고 없음)
- 가격 정보는 "현재 최저가 기준" 표현 사용
- 의약품·건강기능식품 효능 과장 표현 금지 (심의 위반)
